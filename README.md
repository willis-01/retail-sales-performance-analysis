# retail-sales-performance-analysis
Analyze retail sales data to identify trends, top-performing products, customer behavior, and regional performance.
This project analyzes retail sales data from a Superstore dataset to uncover sales trends, customer purchasing behavior, regional performance, and product profitability insights using SQL.
The objective is to demonstrate data cleaning, exploratory data analysis (EDA), business intelligence reporting, and SQL querying skills commonly required for Data Analyst roles.
